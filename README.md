# DataStructures

## Array
  It is a collections of data type stored contiguous memory locations(ordered collections of values)
  
      - Insertion : O(n)
      - Deletion : O(n)
      - Look up: O(n)
      - Append : O(1)

## Hash Table
  It is unordered data structure that stores pairs of key-value

      - Look up: O(1)
      - Insertion: O(1)
      - Deletion: O(1)
